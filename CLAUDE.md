## Agent skills

### Issue tracker

New issues and PRDs are tracked in GitHub Issues for `frgul006/wedding-planner`; existing local issue/PRD files are historical/reference material. See `docs/agents/issue-tracker.md`.

### Triage labels

Use the canonical five-label triage vocabulary: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, and `wontfix`; helper labels include `prd` and `agent-in-progress` (`ready-for-agent` is for implementation issues, not PRDs). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout: root `CONTEXT.md` and root `docs/adr/` when present. See `docs/agents/domain.md`.
